# Network Upgrade Overview

This section has information on how to upgrade your Mainnet and Testnet nodes for new network upgrades.

## Activations

<Callout type="info">
  Network upgrade are activated by timestamps. Failing to upgrade your node
  before the timestamp will cause a chain divergence. You will need to
  resync your node to reconcile the chain.
</Callout>

## Upgrade Process

<Steps>
  ### Devnet

  *   Devnet Upgrade Notice Period
  *   Upgrade Activates on Devnet
  *   Baking on Devnet

  ### Testnet

  *   Testnet Upgrade Notice Period
  *   Upgrade Activates on Testnet
  *   Baking on Testnet

  ### Mainnet

  *   Governance Voting Review Period
  *   Governance Voting Period
  *   Cut Mainnet Release
  *   Mainnet Upgrade Notice Period
  *   Upgrade Activated
</Steps>
